HTML & CSS Cheat Sheet

HTML Tables

Name Tag Description
Table <table></table> Defines a table
Table Row <tr></tr> Defines a row in a table
Table Header <th></th> Defines a header cell in a table
Table Data <td></td> Defines a data cell in a table

Other HTML Tags

Name Tag Description
Paragraph <p></p> Defines a paragraph
Image <img src="" alt=""> Embeds an image
Anchor <a href=""></a> Creates a hyperlink

CSS Selectors and Specificity

Name Selector Description
Element Selector element Selects an element type
Class Selector .class Selects elements with a specific class attribute
ID Selector #id Selects an element with a specific ID attribute
Universal Selector * Selects all elements

Common CSS Properties

Name Property Description
Color color Sets the text color
Background Color background-color Sets the background color
Font Family font-family Sets the font type
Font Size font-size Sets the font size
Font Weight font-weight Sets the font weight (bold, normal, etc.)
Text Align text-align Sets the text alignment (left, right, center, justify)
Padding padding Sets the padding (space inside an element)
Margin margin Sets the margin (space outside an element)
